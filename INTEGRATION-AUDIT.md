### How It Works
1. **IMAP Connection**: Connects to Gmail IMAP3 (port 993 with TLS)
2. **Email Scan**: Fetches emails since specified date (default: 60 days)
3. **AI Classification**: Uses `emailAgent.js` to classify documents
4. **Storage**: Saves to `documents.json` + `entities.json`
5. **Metadata Extraction**: Pulls invoice amounts, dates, senders

### Test Command
```bash

# Expected: { "success": true, "message": "Gmail connected", "user": "info@chickenpalace.es" }
```

### тЪая╕П Known Issues
- **App Password Requirement**: Gmail no longer supports simple passwords. Must use "App Password" (16-char generated by Google)
- **Connection Reset**: If sync takes >5min, Gmail may close connection. Worker handles this with retry logic.
- **Timezone**: Emails stored in UTC. Frontend should convert to local time.

### Next Steps
- [ ] Implement incremental sync (only new emails since last sync)
- [ ] Add filtering by sender domain
- [ ] Support IMAP folders beyond INBOX
