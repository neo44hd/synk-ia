# 🚀 Mission Control — Mando a Distancia SynK-IA\n\n## Acceso Rápido\n\n### Panel Completo (Recomendado)\n```\nhttp://localhost:3001/mc.html\n```\n\n### Launcher con Accesos Rápidos\n```\nhttp://localhost:3001/launcher.html\n```\n\n## Descripción\n\nMission Control es tu panel de control centralizado para gestionar todo el ecosistema SynK-IA desde un único lugar. Es como tener un **mando a distancia** para toda la infraestructura.\n\n## 📊 Pestaña: Dashboard\n\n- **Resumen del sistema**: RAM, CPU, Disco, Uptime\n- **Métricas en tiempo real**: Gráficos de uso de recursos\n- **Estado de servicios**: Docker, PM2, Ollama, Agentes IA\n- **Acciones rápidas**: Reinicio de PM2, Docker, Ollama\n- **Estado de agentes IA**: Verificación rápida de Hermes, OpenCode, etc.\n\n## 🐳 Pestaña: Docker\n\n- **Lista de contenedores**: Estado, puerto, imagen\n- **Acciones**: Restart, Stop, Ver logs\n- **Operaciones por lotes**: Restart all, Stop all, Prune, Compose up/down\n- **Búsqueda y filtrado**: Encontra rápidamente contenedores\n\n## 💚 Pestaña: Agentes\n\n- **Agentes especializados**: Hermes, OpenCode, clasificadores\n- **Estado en tiempo real**: Online, degradado, offline\n- **Acciones**: Restart, Test, Ver logs\n- **Herramientas de desarrollo**:\n  - Claude Code CLI\n  - Aider con Gemma4, Qwen2.5\n  - OpenCode\n  - Codex\n- **Comandos rápidos**: Ollama, PM2, Tailscale, df, top\n\n## 🔄 Pestaña: Pipeline IA\n\n### Fast Path (Rápido)\n- Clasificación de documentos\n- Extracción ultra-rápida\n- Modelos: hermes-9b → qwen2.5-coder\n\n### Deep Path (Profundo)\n- Análisis de documentos complejos\n- Documentos legales, contables, RRHH\n- Modelos: hermes-9b → negentropy-4.7b\n\n### Test Interactivo\n- Tests rápidos de cada path\n- Tester personalizado con endpoints\n- Pruebas de modelos específicos\n\n## 📄 Pestaña: Documentos\n\n- **Gestión de documentos**: Lista de archivos procesados\n- **Estadísticas**: Total, procesados, pendientes, errores\n- **Acciones**: Re-indexar, limpiar procesados, refrescar\n- **Búsqueda**: Filtra por nombre\n\n## 🔌 Pestaña: Servicios\n\n- **Todos los servicios**: Docker, PM2, custom\n- **Información**: Puerto, estado, imagen, PID, RAM/CPU\n- **Acciones**: Logs, Restart, Stop\n- **Búsqueda y filtrado**: Encuentra rápidamente servicios\n\n## 🖥️ Pestaña: Terminal Web\n\n- **Terminal integrada**: Ejecuta comandos directamente\n- **xterm.js**: Terminal completa con soporte de colores\n- **Copiado/pegado**: Ctrl+C para copiar, Ctrl+V para pegar\n\n## 🦙 Pestaña: Ollama\n\n- **Modelos cargados**: Lista de modelos con tamaño y fecha\n- **Acciones rápidas**: \n  - Pull Qwen2.5 14B\n  - Pull Llama 3.3 70B\n  - Pull DeepSeek R1 8B\n  - Eliminar modelos\n- **Test de modelos**: Prueba cada modelo\n- **Búsqueda**: Filtra por nombre\n\n## 🌐 Pestaña: Red\n\n### Tailscale\n- **Estado de conexión**: Conectado/desconectado\n- **Acciones**: Up, Down, Ver estado\n- **IP y estado**: Red privada\n\n### Cloudflare Tunnel\n- **Estado del túnel**: Activo/inactivo\n- **Acciones**: Restart tunnel\n- **Logs**: Estado del túnel público\n\n### Puertos en Uso\n- **Listado de puertos**: LISTEN con aplicación\n- **Información**: Protocolo, PID\n\n## 🔧 Pestaña: Reparación\n\nKit de reparación automática con herramientas:\n\n### Acciones Rápidas\n- 🐳 Restart Docker All\n- 💚 Restart PM2 All\n- 🧹 Docker Prune\n- 🌐 Flush DNS\n\n### Operaciones de Mantenimiento\n- Re-indexación de memoria\n- Reinicio de Ollama\n- Limpieza de DNS\n- Y más...\n\n### Log de Acciones\n- Historial de operaciones ejecutadas\n- Timestamps y resultados\n\n## 🔐 Autenticación\n\n**Token de acceso**: `sinkia2026`\n\nEl token está integrado en el panel.
